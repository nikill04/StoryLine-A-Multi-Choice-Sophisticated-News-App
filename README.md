##  A Multi-Category News Application

This is a multi-choice news application that allows users to read news from
multiple categories such as entertainment, health, politics, sports, and more.

The application fetches real-time news content from an external API and displays
it in a clean and user-friendly interface, focusing on readability and ease of use.

### Getting Started

To run the project locally: `npm start`

### Environment Setup

The application uses an API key to fetch news data.

Create a file named .env.local in the project root directory and add:

REACT_APP_NEWS_API=your_api_key_here
